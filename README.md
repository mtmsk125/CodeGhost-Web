# CodeGhost 👻 - AI Bug Fixer

موقع ببلاش بصلحلك الـ Bugs تبعت الكود بـ 3 ثواني باستخدام Gemini AI.

## كيف تستخدمه؟
1. لزق الكود اللي فيه Error بالمربع الأول
2. لزق رسالة الخطأ الحمرا كاملة بالمربع الثاني  
3. دوس "صلحلي ياه يا شبح 🔧"
4. انسخ الكود المصح واستخدمه

## التقنيات المستخدمة
- Frontend: HTML + CSS + JavaScript
- Backend: Node.js + Express
- AI: Google Gemini 2.5 Flash API

## التشغيل Local
